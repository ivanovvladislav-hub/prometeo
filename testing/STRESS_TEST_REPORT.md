# Отчёт о стресс-тестировании

**Дата:** 13 марта 2026
**Студент:** Иванов Владислав

## 1. Конфигурация
*   Виртуальная машина: VirtualBox, 2 ядра, 4GB RAM.
*   API: FastAPI, запущено через uvicorn.
*   База данных: PostgreSQL в Docker.

## 2. Результаты тестов

### Тест 1: Лёгкая нагрузка (10 пользователей)
*   Время работы: 2 минуты.
*   Ошибок: 0.
*   Среднее время ответа: ~2 мс.
*   Вывод: API работает отлично.

### Тест 2: Средняя нагрузка (50 пользователей)
*   Время работы: 2 минуты.
*   Ошибок: 51 (только на GET /api/stats/1, POST работали без ошибок).
*   Среднее время ответа POST: ~2 мс.
*   Вывод: Эндпоинт статистики не работает, но отправка данных стабильна.

### Тест 3: Высокая нагрузка (200 пользователей)
*   Время работы: 2 минуты.
*   Ошибок: начали появляться при росте нагрузки.
*   Среднее время ответа: ~350 мс.
*   Нагрузка на CPU: 40%, MEM: 1.36 GB.
*   Вывод: Система выдерживает, но требует доработки эндпоинтов.

## 3. Выводы
Разработанная система стабильно обрабатывает отправку данных. Основная проблема  неработающий эндпоинт статистики. Для улучшения производительности нужно исправить API и добавить кэширование.
