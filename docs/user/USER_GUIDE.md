## 1. Введение

### 1.1. Назначение системы

Prometeo — это система мониторинга температуры в режиме реального времени. Она позволяет:
- Отслеживать показания с множества датчиков
- Визуализировать данные на графиках
- Получать статистику за различные периоды
- Экспортировать данные для отчётов

### 1.2. Требования к рабочему месту

- **Браузер:** Google Chrome 90+, Firefox 88+, Safari 14+ или Edge 90+
- **Разрешение экрана:** минимум 1280x720 (рекомендуется 1920x1080)
- **Подключение к сети:** доступ к серверу Prometeo

## 3. Основные операции

### 3.1. Выбор датчика для просмотра

1. В верхней части найдите выпадающий список **"Выберите датчик"**
2. Кликните на него
3. Выберите нужный датчик из списка (например, "Sensor-001 (Server Room A)")
4. Данные автоматически обновятся

### 3.2. Изменение временного диапазона

Над графиком расположены кнопки периодов:

- **1ч** — данные за последний час
- **6ч** — за 6 часов
- **24ч** — за сутки (по умолчанию)
- **7д** — за неделю

**Действия:**
1. Кликните на нужную кнопку
2. График и статистика пересчитаются автоматически

### 3.3. Чтение статистических карточек

В верхней части дашборда отображаются 4 карточки:

| Карточка | Описание |
|----------|----------|
| **Среднее значение** | Средняя температура за выбранный период |
| **Минимум** | Самое низкое значение |
| **Максимум** | Самое высокое значение |
| **Записей** | Количество измерений в базе |

**Цветовая индикация:**
- 🔵 Синий — среднее значение
- 🟢 Зелёный — минимум
- 🔴 Красный — максимум
- 🟣 Фиолетовый — количество данных

### 3.4. Работа с графиком

**Интерактивные функции:**

1. **Наведение курсора** — показывает точное значение и время
2. **Масштабирование** — прокрутка колёсика мыши (если включено)
3. **Легенда** — клик по названию в легенде скрывает/показывает линию

**Интерпретация графика:**
- Горизонтальная ось (X) — время
- Вертикальная ось (Y) — температура в градусах Цельсия
- Данные агрегируются каждые 5 минут для оптимизации производительности

### 3.5. Экспорт данных

Для сохранения данных в виде таблицы:

1. Нажмите кнопку **"📥 Экспорт"** над графиком
2. Браузер автоматически скачает CSV-файл
3. Файл можно открыть в Excel, Google Sheets или любом текстовом редакторе

3.6. Обновление данных
Автоматическое обновление:

Данные обновляются каждые 30 секунд автоматически

Индикатор подключения (зелёная точка) показывает статус системы

Ручное обновление:

Нажмите кнопку 🔄 в правом верхнем углу

Иконка начнёт вращаться

Данные обновятся немедленно

4. Поиск датчиков
Если в системе зарегистрировано много датчиков:

Введите название или расположение в поле "Поиск датчиков..."

Выпадающий список автоматически отфильтруется

Выберите нужный датчик

Примеры запросов:

Server — покажет все датчики в серверных

Room A — датчики в комнате A

002 — датчик с ID 002

5. Информация о датчике
В нижней части дашборда отображается детальная информация:

Поле	Описание

Статус	🟢 Активен / 🔴 Неактивен


6. Решение типовых проблем
6.1. График не отображается
Причина: Нет данных за выбранный период

Решение:

Выберите более длинный период (например, 7 дней вместо 1 часа)

Проверьте, что датчик активен (статус 🟢)

Обновите страницу (F5)

6.2. Данные не обновляются
Причина: Потеря подключения к серверу

Решение:

Проверьте индикатор статуса (должен быть зелёным)

Проверьте подключение к сети

Обновите страницу

6.3. Медленная загрузка
Причина: Большой объём данных

Решение:

Уменьшите временной период

Очистите кэш браузера (Ctrl+Shift+Del)

Обратитесь к администратору для оптимизации
